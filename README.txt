Приложение для обмена зашифрованными ссылками.

Используемые технологии: 
	Логика и фронт: ReactJS.
	Бэкэнд: NodeJS.
	БД: MySQL.

Как пользоваться.
	Во вкладке Home можно создать либо просмотреть уже созданный note.
	Во вкладке Create можно создать note? введя текст сообщения в поле и нажав кнопку "Создать", после чего на экране 	появится хэш заметки. Его нужно скопировать и вставить в поле во вкладке Note и нажать кнопку "Искать Note".
	На экране появится текст зашифрованного сообщения.
